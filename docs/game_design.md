# GAME_DESIGN.md - IncriElemental Game Mechanics

This document defines the mechanics, balance formulas, and progression for **IncriElemental**.

## 1. Visual Progression
The game's atmosphere is a core feedback loop. As elemental essences are mastered, the "Aetherial Glow" intensifies:
- **Early Game:** Deep black void with dim, scattered stars.
- **Mid Game:** Nebula clouds emerge; UI borders begin to pulse with elemental energy.
- **Late Game:** High-intensity bloom and global color grading reflect the dominant essence.

## 2. Manifestation System
Manifestations are the primary driver of progression, utilizing a component-driven design:
- **Producer:** Generates resource per second.
- **Storage:** Increases the maximum cap of a resource.
- **Aura:** Projects influence onto neighboring cells on the World Map.
- **Unlock:** Discovers new manifestations or tabs.

## 3. World Map & Biomes
The World Map is a grid-based spatial puzzle:
- **Aura Synergies:** Placing structures creates influences that boost neighboring production.
- **Biome Animations:** Map cells feature refractive Ocean ripples and heat-haze Mountain animations to distinguish biomes visually.
- **Aura Pulses:** Active influences are visualized via animated, pulsing cell borders.

## 4. Alchemical Mixing
The Mixing Table allows for active player experimentation:
- **Reaction Sequences:** Successful mixes trigger high-impact visual sequences (radial blur, particle bursts).
- **Buff Feedback:** Active buffs are displayed in the Status Bar with unique glowing "medals."

## 5. Ascension & NG+
Ascension is the ultimate milestone, resetting the loop for permanent "Cosmic Insight" multipliers. This transition is marked by a "Pure White" visual celebration sequence.
